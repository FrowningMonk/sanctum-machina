# Task 6: AboutScreen + navigation entry + ModelManagerScreen TopAppBar action + `SafeMarkdown`

## Description

Реализуем экран «О приложении» (`AboutScreen`) с безопасным рендерингом Markdown и точкой входа из `ModelManagerScreen`.

`SafeMarkdown.kt` — Composable-обёртка над `RichText { Markdown(text) }` из библиотеки compose-richtext (решение D8). Внутри использует `SafeUriHandler` — кастомный `UriHandler`, который разрешает переходы только по схемам `http` и `https`, все остальные схемы (`intent`, `sms`, `tel`, `javascript`, `file`, `content`, `data`, `market` и т.д.) молча блокирует (решение D25). Это защита от URI injection через вредоносный Markdown-контент.

`AboutScreen.kt` — scrollable Composable (решение D17). Читает файл `assets/about.md` через `context.assets.open("about.md")` (имя файла захардкожено — защита от path traversal). Рендерит содержимое через `SafeMarkdown`. В футере показывает `BuildConfig.VERSION_NAME` и текст атрибуции.

Навигация: в `NavHost` добавляется destination `"about"`. На `ModelManagerScreen` в `TopAppBar` добавляется `IconButton` с иконкой `Icons.Default.Info`, который вызывает `navController.navigate("about")`.

**ВАЖНО:** `app/src/test/` директория не существует и в `app/build.gradle.kts` нет `testImplementation` блока. Перед написанием `SafeUriHandlerTest` необходимо добавить `testImplementation(libs.junit)` (и при необходимости `testImplementation(libs.truth)` или `testImplementation(libs.kotlin.test)`) в `app/build.gradle.kts`. `ModelManagerScreen` использует callback-паттерн (не navController напрямую) — добавить `onAbout: () -> Unit` callback по аналогии с существующим `onLoad`

**Edge cases:**
- `about.md` отсутствует в assets → показывать fallback-строку, не крашиться
- Malformed URI в Markdown → `SafeUriHandler` перехватывает и молча блокирует (не кидает exception)
- `BuildConfig.VERSION_NAME` пустой или null → использовать строку-заглушку

**Implementation hints:**
- `SafeUriHandler` реализует интерфейс `UriHandler` из Compose; в методе `openUri` парсить схему через `Uri.parse(uri).scheme`, проверять whitelist.
- Передавать `SafeUriHandler` через `CompositionLocalProvider(LocalUriHandler provides safeHandler)` перед вызовом `RichText { Markdown(...) }`.
- Имя файла `about.md` хранить в `private const val ABOUT_ASSET = "about.md"` — никогда не принимать из параметров.
- Чтение assets в Composable — через `LocalContext.current` + `remember { context.assets.open(...).bufferedReader().readText() }` с try/catch.
