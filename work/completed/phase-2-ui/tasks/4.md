# Task 4: `MultimodalContentsBuilder` + рефакторинг `LlmChatModelHelper.runInference`

## Required Skills

Перед выполнением задачи загрузи:
- `/skill:code-writing` — [skills/code-writing/SKILL.md](~/.claude/skills/code-writing/SKILL.md)

## Description

Вынести inline-сборку `List<Content>` из `LlmChatModelHelper.runInference` в отдельный pure-класс `MultimodalContentsBuilder.build`. Функция `runInference` обрабатывает images/audio перед вызовом `sendMessageAsync` — эта логика должна жить в изолированном, тестируемом месте, а не внутри сложного метода.

Рефакторинг по решению D22 ([TECHNICAL]): никакого изменения поведения, только разделение ответственности.

## What to do

1. Прочитать `LlmChatModelHelper.runInference` — найти inline-сборку `List<Content>` перед `sendMessageAsync`.
2. Создать `MultimodalContentsBuilder` в `:core-runtime/common/` с методом `build(text, images, audio)`.
3. Перенести логику сборки в `build`, соблюдая порядок: `ImageBytes`, `AudioBytes`, `Text`.
4. Пустой текст → `Content.Text` не добавлять.
5. Заменить inline-сборку в `runInference` вызовом `MultimodalContentsBuilder.build`.
6. Написать unit-тесты до реализации (TDD).

## TDD Anchor

Тесты пишем ДО реализации — запускаем, убеждаемся что падают, потом пишем код.

- `MultimodalContentsBuilderTest::textAndImagesAndAudio_correctOrder` — `(text, listOf(bitmap), listOf(bytes))` → `listOf(Content.ImageBytes, Content.AudioBytes, Content.Text)` в таком порядке
- `MultimodalContentsBuilderTest::emptyText_noTextContent` — пустой текст → `Content.Text` не добавляется
- `MultimodalContentsBuilderTest::textOnly_singleTextContent` — `(text, emptyList, emptyList)` → ровно один `Content.Text`
