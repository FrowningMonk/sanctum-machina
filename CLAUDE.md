# Проект: PhoneWrap

> **Форк Google AI Edge Gallery с добавлением истории чатов, проектов и общего контекста для локальных LLM на мобильных устройствах.**

---

## Как устроен проект

**Контекст:** вся база знаний проекта лежит в скилле `.claude/skills/project-knowledge/` — там гайды по архитектуре, паттернам, деплою (+ опционально UX-гайдлайны и доменные файлы).

**UI-дизайн:** источник правды по визуалу — `docs/design_handoff_phase3_ui/`. При любых правках UI сверяться с `README.md` и токенами оттуда.

**Ветка по умолчанию:** `main` (git ещё не инициализирован — инициализируем при необходимости).

**Документация библиотек:** Context7 MCP не используется. Для получения документации внешних библиотек — WebFetch / WebSearch / gh CLI напрямую.

---

## Язык общения

- С пользователем, в планах, в plan-mode: **русский**.
- Код и комментарии в коде: английский.
- Промпты для LLM: английский.
- Документация для пользователя (README): русский.
- Служебные файлы фреймворка (скиллы, агенты, команды) остаются как есть — менять их язык нет смысла.

## Поведение

- Без «Отличный вопрос!», без воды, без филлеров.
- НИКОГДА не использовать инструмент AskUserQuestion. Задавай вопросы в чате обычным текстом.
- Деплой не настроен (мобильное приложение). Не предполагай наличие CI/CD.

## Планирование задач

- Для многошаговых задач (>1 шага) веди to-do список через инструменты `TaskCreate` / `TaskUpdate` / `TaskList` / `TaskGet`.
- Помечай задачу `in_progress` перед началом работы и `completed` сразу после завершения.
- Для запуска субагента (валидатора, ревьюера, исследователя) используй инструмент `Agent`.

## Безопасность

- НИКОГДА не проси пользователя писать секреты в чат.
- Вместо этого — даю инструкции, куда их безопасно положить:
  - Локально: `.env`, конфиг-файлы.
  - CI/CD: GitHub Actions secrets (когда CI будет настроен).
- ВСЕГДА спрашивать перед деплоем / push в main / production.
- ВСЕГДА добавлять секреты в `.gitignore`: `.env`, `*.key`, `credentials.json`, `secrets/`.
- Осторожно с внешними действиями (push, deploy, отправка сообщений, создание PR). При сомнениях — спросить.
