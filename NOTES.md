## 🔴 Большая красная кнопка

**Последнее гарантированно рабочее состояние** — `v0.1-foundation` (merge-коммит Phase 1 на `main`, SHA `60dcb4c`).

Как откатиться, если что-то всё сломали:

```bash
git reset --hard v0.1-foundation     # вернуться на Phase 1 (последняя одобренная точка)
git reset --hard master              # ядерный сброс на самое начало (пустой проект + gitignore)
```

`main` — тоже «безопасное место», т.к. туда попадают только одобренные мной фазы (правило: merge в main только после явного «утверждаю»). После старта Phase 2 главная красная кнопка останется на `v0.1-foundation`, пока Phase 2 не будет одобрена — тогда появится `v0.2-ui` и станет новой точкой отсчёта.

Теги `master` / `v0.X-*` — immutable. Никогда не перевешиваем.

### Backlog для будущих фаз

- **Free-scroll во время стрима** (Phase 5 polish, ~30 строк, изолировано в `ChatScreen.kt`). Сейчас автоскролл всегда тащит ленту вниз; хочется sticky-to-bottom — пока пользователь у самого низа, лента следует за стримом, как только он сам проскроллил вверх — автоскролл замирает, можно спокойно читать историю. Возврат к низу (вручную или по плавающей кнопке-стрелке «↓») возобновляет автоскролл. Идея пришла во время smoke task 11 (2026-04-18).
